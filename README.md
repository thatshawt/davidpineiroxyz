# davidpineiro.xyz
This is the source code for my website.
Its live at [https://davidpineiro.xyz](https://davidpineiro.xyz).

## devops features
* Continuous Deployment using NixOS and GitHub.

## tech stack
* Redbean 3.0.0, an open source webserver in a single-file that runs natively on six OSes for both AMD64 and ARM64.
* Fullmoon, Fast and minimalistic Redbean-based Lua web framework in one file.
