Live Demo Link

Summary

I was given a front end coded in angular and was tasked with designing a REST api back end using spring boot. I followed the instructions, reading the front end code to help understand what to put into the DTO classes and to understand the endpoints in general.

To deploy the live demo I configured an nginx reverse proxy and provided SSL with nginx as well. The database, front end, and back end are all configured with NixOS as various systemd services/units.
